TRIM VIDEO - Složka pro zkracování videí
==========================================

1. Umístěte sem dlouhé video (MP4, AVI, MOV, MKV)

2. Spusťte: python trim_video.py

3. Zkrácené video (1 minuta) se objeví ve složce 'input/'
   a je připravené pro analýzu

4. Poté spusťte: python process_video.py
   nebo: python process_video_advanced.py
